A linguagem JavaScript permite agrupar funções e dispô-las em um arquivo separado a da página HTML. Isso traz muitos benefícios:
1-Reutilização de funções em muitos arquivos.
Não tempos que copiar e colar sucessivamente as funções nas páginas que necessitamos.
2-Facilita a manutenção das funções ao se encontrar em arquivos separados.
3-Nos obriga a ser mais organizados.



A mecânica para implementar estes arquivos externos no JavaScript é:
1-Criar um arquivo com extensão *.js e tipar as funções na mesma;
2-Criar um arquivo html que utilizará as funções contidas no arquivo *.js;
3-Dispor o seguinte código para importar o arquivo *.js:
<script type="text⁄javascript" scr="funcoes.js><⁄script>


-Mediante a propriedade src indicamos o nome do arquivo para importar. -Logo, podemos chamar dentro da página HTML, as funções que contém o arquivo externo *.js; em nosso exemplo chamamos as funções retornaData() e retornaHora().
-O arquivo html fica muito mais limpo.
